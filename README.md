# 🏠 Real Estate Price Prediction

## 📌 Project Overview
This project predicts real estate prices based on property features using a Machine Learning model. The application is built with Python and Streamlit, allowing users to enter property details and receive an estimated price instantly.

---

## 🚀 Features
- Predicts real estate prices
- Simple and interactive Streamlit interface
- Uses a trained Machine Learning model
- Accepts user input for:
  - Total Square Feet
  - Number of Bathrooms
  - Number of Balconies
  - Number of BHK

---

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- Joblib

```

```
